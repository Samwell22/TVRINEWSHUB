# Folder Videos

Folder ini digunakan untuk menyimpan file video berita (opsional).

**Format yang diizinkan**: MP4, WEBM, AVI
**Ukuran maksimal**: 10MB (untuk demo, bisa dinaikkan untuk production)
**Rekomendasi**: Gunakan YouTube embed untuk video besar

Folder ini dilindungi oleh `.htaccess` untuk mencegah eksekusi file PHP (keamanan).
