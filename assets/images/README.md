# Folder Images

Folder ini untuk menyimpan asset gambar website seperti:

- **Logo** TVRI Sulut
- **Favicon** website
- **Default thumbnail** untuk berita tanpa gambar
- **Icon** dan grafis lainnya

## File yang Diperlukan:

- `default-news.jpg` - Placeholder untuk berita tanpa thumbnail (800x600px)
- `logo.png` - Logo TVRI Sulawesi Utara
- `favicon.png` - Icon browser tab

**Note**: Untuk sementara, jika gambar belum tersedia, sistem akan tetap berjalan. Tambahkan file-file gambar sesuai kebutuhan nanti.
